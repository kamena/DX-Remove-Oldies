== Description ==

Delete the posts from Custom Post Types that are no longer active for the install.

Navigate to Settings -> Remove Oldies and delete the posts from a given post type that is not active anymore.

== Installation ==

1. Upload the DX Remove Oldies plugin to your blog and activate it.
2. Go to Settings -> Remove Oldies
3. Click 'Delete' for any post type from the table that you will no longer use.
